# 便利Slack Bot

## About This

一般的に便利と思われる機能を追加していくSlack Bot。汎用性を追い求めていくぞ。
## Slack App Require scopes

* Oauth & Permissions
  * Scopes -> Bot Token Scopes
    * `chat:write`
    * `channels:read`
    * `emoji:read`
* Event Subscriptions
  * Subscribe to bot events
    * `emoji_changed`
    * `channel_created`
    * `channel_rename`
    * `channel_archive`
    * `channel_deleted`
